# Kaiju City

This is a web game I made using React, the UI library.

* Kaiju City is a top-down, real-time strategy game where the player uses his abilities and positioning on the gameboard to defeat the godzilla-like monsters and save the city. The game features:
    - emergent gameplay based on tile statuses
    - an enemy and teammate AI
    - a short tutorial
    - AI-generated art assets

## Getting Started

1. Clone the repo locally.
2. In your terminal, navigate to the main folder of the cloned repo and type:
```npm install```
**If you are having issues npm-installing, remove ```  "type": "module", ``` from your package.json (second line) and try installing again. (Just be sure to add the line back after install.)**
3. Once installed, type:
```npm start```
and open http://localhost:9000/ in your browser.

### Prerequisites

You will need the node and npm to play.

### Note

During gameplay, feel free to zoom-in/zoom-out the browser content (Ctrl and +/- on Windows, Cmd and +/- on Mac), for a better viewing experience.
